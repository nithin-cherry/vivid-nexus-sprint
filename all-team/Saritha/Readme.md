# NorthGate Advisory — landing page

A sample business landing page built with React + Vite.

## Folder structure

```
fenwick-cole-landing/
├── index.html
├── package.json
├── vite.config.js
├── README.md
└── src/
    ├── main.jsx           # React entry point
    ├── App.jsx            # Root component
    ├── index.css          # Global resets
    └── components/
        └── LandingPage.jsx  # The full landing page
```

## Getting started

```bash
npm install
npm run dev
```

Then open the local URL Vite prints (usually `http://localhost:5173`).

## Build for production

```bash
npm run build
npm run preview
```
