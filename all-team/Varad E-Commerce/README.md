# CrickForge — Premium Cricket Store

React + Vite storefront, blue/white/willow theme.

## Run locally
```bash
npm install
npm run dev
```

## Deploy to Vercel

**Option A — CLI**
```bash
npm i -g vercel
vercel
```

**Option B — Dashboard**
1. Push this folder to a GitHub repo
2. [vercel.com/new](https://vercel.com/new) → import the repo
3. Framework preset: **Vite** (auto-detected) → Deploy

No env vars needed.
